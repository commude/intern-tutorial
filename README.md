# laravel-todo

## Requirements
- PHP 8.1.2
- Laravel 9.7.0
- Any database system
- Composer 2.x
- Npm 7.21 (or anything near) & yarn
- Other PHP extension

## Environment
- Docker & Docker Compose
- Nginx 
- MySQL 5.7

## Setup
1. Build docker image first.

```bash
cd environments/local
cp .env.example .env
docker-compose build
```

2. Create Containers

```bash
docker-compose up -d
```

3. Entering the Container&Install libraries

```bash
docker exec -it laravel-todo.php-fpm bash
```

local %  →→　　bash-4.2#

```bash
composer install
```

4. Make & edit .env

```bash
cp .env.example .env
php artisan key:generate
```

Edit the .env under src folder
